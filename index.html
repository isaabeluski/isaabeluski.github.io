Human Activity Recognition & Sleep Classification
Python, TensorFlow — Sep 2024 – Apr 2025
  • Objective: Develop wearable-sensor-based methods to assess sleep quality in COPD patients using the RESpeck device.
  • Hierarchical HAR pipeline:
    • Static vs. dynamic activity classifier: CNN achieved 97% test accuracy.
    • Fine-grained static posture model: 95% accuracy distinguishing back/side/stomach positions.
    • Dynamic activity model with transfer learning: improved from 50.7% → 71% accuracy.
  • Sleep episode classification using FAMD + K-Means on combined HAR labels & continuous features to segment sleep/wake.
  • Extracted nightly sleep metrics (duration, efficiency, latency, WASO, position changes) and correlated with cough frequency & CAT scores.
  • Key findings: Supine sleep correlates with more nocturnal coughing; significant positive correlations between sleep efficiency, latency, positional changes, and CAT symptom score (p < 0.05).

IoT Systems for Live Activity Classification
Python, Deep Learning, Machine Learning, Android Studio — Jan 2025
  • Objective: Build, in a team of 3, an IoT activity-recognition system classifying eleven physical activities (stairs, running, walking, shuffle, lying in four orientations, sitting/standing, miscellaneous) and four respiratory states (coughing, hyperventilating, normal, other) via wearable RESpeck & Thingy sensors.
  • Methods:
    • 1D CNNs on RESpeck data for eleven activities; separate CNN on Thingy data to distinguish sitting vs. standing.
    • Evaluated LSTM, SVM, Random Forest—1D CNN proved most accurate (>90% static, >80% respiratory).
    • Android app (Kotlin + TensorFlow Lite) for real-time classification, data recording, visualisation, and history.
  • Results:
    • Cross-validated CNNs delivered >90% accuracy on static/dynamic activities and >80% on respiratory states.
    • Real-time app classification averaged 99% for physical activities and 84% for respiratory conditions.

Computer Vision: Semantic Segmentation
Python, PyTorch, UNet, CLIP, Gradio — April 2025
  • Objective: Compare segmentation approaches (UNet, autoencoder pretraining, CLIP-based features, prompt-based model) on the Oxford-IIIT Pet Dataset.
  • Preprocessing & Augmentation:
    • Resize images/masks to 224×224, normalise with ImageNet stats.
    • Augment with flips, rotations, random crops, elastic transforms, grid distortions.
  • Models evaluated:
    • UNet trained with weighted cross-entropy — achieved mean IoU ~0.73, Dice 0.78, pixel accuracy 0.91.
    • Autoencoder pretraining (simple, improved, UNet-based) — simple AE IoU ~0.57, improved AE IoU ~0.37, UNet-AE IoU ~0.44.
    • CLIP-based segmentation using ViT features — mean IoU ~0.83, Dice 0.91, pixel accuracy 0.93.
    • Prompt-based model with CLIP + Gaussian point prompts — mean IoU ~0.89, Dice 0.94, pixel accuracy 0.94; interactive Gradio demo built.
  • Robustness: Prompt-based model remained above 0.80 Dice under noise, blur, contrast, occlusion.
  • Key takeaway: CLIP features + prompt conditioning deliver top segmentation quality and interactivity, outperforming traditional architectures.

Enhancing Sheep Detection Techniques with Deep Learning
Python, TensorFlow, SSD — March 2024
  • Objective: Automate sheep counting and detection using deep learning on a new diverse video dataset considering occlusion and sheep behaviours.
  • Dataset: 149,327 frames from 417 videos labelled for five activities (grazing, running, walking, standing, sitting) and occlusion status.
  • Method: Transfer learning on SSD300-VGG16 pretrained on COCO dataset, with experiments in data augmentation (cropping) and optimisation (SGD, Adam).
  • Results: SSD model consistently outperformed Faster R-CNN across IoU thresholds; fine-tuned approaches improved MAE/MSE and true positives but faced challenges under heavy occlusion.
  • Key takeaway: Behavioural and contextual factors critically affect detection accuracy; future work to explore advanced schedulers, model architecture changes, and larger annotated sets.
